### Find the probability of getting exactly two "head" when flipping three coins.

|      |      |
| ---- | ---- |
| HHH  | THH  |
| HHT  | THT  |
| HTH  | TTH  |
| HTT  | TTT  |

$$
p(2'H') = \frac{3}{8}
$$
